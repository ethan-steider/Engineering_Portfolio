Capstone Project: Modular Drone Platform for Undergraduate Research

Capstone • Aug 2025 – May 2026

System Overview

This project focuses on developing a modular octocopter platform for environmental and climate research. Flight control is handled by a Pixhawk Cube Orange+, while onboard data collection and networking are managed by an NVIDIA Jetson. Optical, thermal, and weather sensors can be operated individually or together to support flexible data collection in field deployments.

Key Engineering Contributions

  • Debugged GPS and telemetry issues to restore autonomous flight and achieve consistent satellite lock.
  • Improved Jetson power delivery to prevent brownouts and reduce boot time.
  • Developed a modular sensing UI (more detail below) supporting optical, thermal, and weather sensors with separate data logging.

Hardware & Architecture

  • Flight Control: Pixhawk Cube Orange+ running ArduPilot.
  • Compute: NVIDIA Jetson Nano (onboard data capture & communications) and Jetson Orin (offline AI processing).
  • Sensing: Optical camera (HDMI→USB), thermal camera (USB), and SCD-41 weather sensor (I²C).
  • Communications: TP-Link CPE210 directional antenna relaying a 2.4 GHz Local Area Network (LAN) for low-latency video and data.

Testing & Results

  • Conducted short flight tests to evaluate hover stability, GPS performance, and system integration.
  • Achieved stable autonomous flight with reliable GPS lock and telemetry link.
  • Collected sample data from sensors during test flights.

Engineering Standards

  • IEEE 1937.1 – Payload interface and performance requirements for drone-mounted sensors.
  • IEEE 1939.1 – Framework for low-altitude airspace and UAV operations.
  • IEEE 802.11 – Wireless LAN standards used for onboard data transmission.

Jetson-Based Video Streaming & Ground Station UI

Capstone Subproject • Primary Contribution

System Overview

Developed an onboard video streaming system and browser-based ground station interface to view live camera feeds from the UAV. The system runs on an NVIDIA Jetson and supports live viewing, recording, and basic operator interaction over a local network as part of the Capstone platform.

Key Engineering Contributions

  • Implemented a Python and Flask-based web server to stream live camera feeds to a browser interface.
  • Integrated optical and thermal USB cameras, including handling device enumeration and swapping.
  • Added recording and file management features for collecting field data.
  • Improved system reliability by addressing startup, power, and reconnection issues observed during testing.

Hardware / Software Stack

  • Hardware: NVIDIA Jetson Nano, USB cameras (optical + thermal)
  • Software: Ubuntu/Linux, Python, Flask, OpenCV, HTML/CSS (same as this Portfolio).

Testing & Results

  • Tested live video streaming during ground and flight operations.
  • Observed stable viewing of camera feeds with low end-to-end latency at short range.
  • Captured representative in-flight video and interface screenshots for documentation.
